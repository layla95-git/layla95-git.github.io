- `_config.yml`
- `Gemfile`
- `index.html`
- `index_layout2.html`
- `publications.html`
- `showcase.html`
- `_data/`
- `_includes/`
- `_layouts/`
- `_publications/`
- `_news/`
- `_showcase/`
- `assets/`

Important paths:

- Profile photo: `assets/images/photos/Profile.jpg`
- Publication thumbnails: `assets/images/covers/`
- Optional CV: `assets/CV.pdf`

GitHub Pages settings:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`
- Do not add `.nojekyll`.
