# Le Li Academic Homepage

This repository contains the source files for Le Li's personal academic homepage, built with a Jekyll academic-homepage style layout.

## Main editable files

- `_data/profile.yml`: name, affiliation, bio, links, education, experience, awards, and academic service.
- `_data/news.yml`: news items shown on the homepage.
- `_publications/`: selected publication cards shown on the homepage and Publications page.
- `_data/co-author_pub.yml`: complete publication list.
- `assets/images/photos/Profile.jpg`: profile photo.
- `assets/images/covers/`: optional publication thumbnails.
- `assets/CV.pdf`: optional CV file. Upload it and enable `cv_link` in `_data/profile.yml`.

## Deployment

Use GitHub Pages with `main` branch and `/root` as the publishing source. Do not add a `.nojekyll` file.
